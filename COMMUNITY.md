# Community & Support

## 💬 Discussions

We use [GitHub Discussions](https://github.com/VyomKulshrestha/Cortex-OS/discussions) for community conversations.

### Discussion Categories

- **💡 Ideas** — Share feature ideas and plugin concepts
- **❓ Q&A** — Ask questions about setup, usage, or development
- **🎉 Show & Tell** — Share your custom plugins, gesture workflows, or configurations
- **🐛 Bug Reports** — For complex OS-specific bugs that need community input
- **📢 Announcements** — Release notes and important updates

### How to Enable GitHub Discussions

1. Go to your repo **Settings** → **General**
2. Scroll to **Features** section
3. Check ✅ **Discussions**
4. Click **Set up discussions**

## 🔌 Sharing Plugins

Created a cool plugin? Share it with the community:

1. Create your plugin in `~/.config/cortex-os/plugins/`
2. Test it thoroughly
3. Share it in the **Show & Tell** discussion category
4. Or submit it as a PR to the `community-plugins/` directory

## 📜 Code of Conduct
